About Sensi Legends

Sensi Legends is an NFT collection highly inspirated in the classic Sensible Soccer. Each NFT is programatically generated and includes manually selected properties according to the competition, team and player and random elements, making each NFT an unique digital collectible.

Each NFT has:

  • Manually selected properties
  • Random properties
  • 100% Programaticaly generated layers

We will do our best to carefully select each NFT's properties, to make sure it has the highest degree of resemblance possible considering the existing variables.
